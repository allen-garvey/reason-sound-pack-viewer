# Reason Sound Pack Viewer

Preview all Reason+ sound packs in your browser.

## Dependencies

* npm
* node >= 14

## Getting Started

* `npm install`
* `npm run hydrate`
* `npm start`

## Make production build

* `npm run deploy`
